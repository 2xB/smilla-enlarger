Loading an Image

As described in 'Drop and Paste', you can import images by Drag&Drop or paste them via the clipboard.

Or choose 'Open...' in the file menu,

or press Ctrl-O / Cmd-O.

Or click 'Open...' at the top of the 'Cropping' tab ( in the same tabbed view as this help ).

After selecting a picture, you will see it displayed in the 'Cropping' tab.
To quickly switch to another file in the same folder, click onto the box containing the filename, you will see that it's a combo-box listing all pictures in the source folder. Supported types for loading are

JPG, GIF, TIF, BMP, PPM and PNG.
